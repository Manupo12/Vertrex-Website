2. Gestión de Inventario y Catálogo

  1. Creación de Productos: Diríjase a la sección Productos > Añadir Nuevo. Es obligatorio diligenciar los campos de Nombre, Precio (COP), Categoría y cargar al menos una fotografía descriptiva.
  2. Gestión de Categorías: Antes de crear productos, asegúrese de tener configuradas las categorías correctas en Ajustes > Categorías para que los clientes puedan filtrar los equipos en la tienda.
  3. Control de Existencias (Stock): El sistema descuenta automáticamente una unidad del inventario cada vez que la pasarela de pagos (Wompi) confirma una transacción exitosa.

3. Módulo de Reparaciones (Servicio Técnico)

  • Ingreso de Equipos: Al recibir un dispositivo, cree un ticket nuevo asociando los datos de contacto del cliente y el diagnóstico preliminar.
  • Transición de Estados: Para mantener al cliente informado, debe cambiar el estado del ticket: "Recibido" → "En Reparación" → "Listo para Entregar".
  • Consulta Externa: Su cliente no necesita llamar; puede ingresar el ID del Ticket en la sección web de soporte para ver en qué estado se encuentra su dispositivo.
