Buena pregunta — y te corrijo algo clave desde ya:

👉 **NO, el oficio NO es lo mismo que la propuesta.**

* **Propuesta** = documento de venta (para cerrar al cliente)
* **Oficio** = documento formal (más institucional, más “legal vibe”)

Si usas ambos bien, subes muchísimo la percepción de valor.

---

# 🧠 PRIMERO: EL SISTEMA COMPLETO VERTREX (VISION GENERAL)

Tú no vas a tener “unos documentos sueltos”.

Vas a tener un **sistema estructurado en 5 fases**:

1. Captación (antes de vender)
2. Cierre (cuando ya va a pagar)
3. Inicio del proyecto
4. Desarrollo / seguimiento
5. Entrega + postventa

Cada fase tiene documentos + portal + comunicación.

# 🟢 FASE 1 — CAPTACIÓN (ANTES DE QUE PAGUE)

## 📄 1. Propuesta comercial (OBLIGATORIO)

**Formato:** PDF (diseñado en Canva)

**Propósito:**

* Vender
* Explicar el valor
* Justificar el precio

**Cuándo se entrega:**
👉 Antes de que el cliente pague
👉 Después de entender lo que necesita

**Contenido:**

* Problema del cliente
* Solución (tu sistema)
* Qué incluye
* Beneficios
* Precio ($400k / $600k etc)
* Forma de pago (50% anticipo)
* Timeline
* Call to action
